# Arif Ritzky Baridi - Personal Portfolio

Selamat datang di repositori portofolio pribadi saya. Website ini dibangun untuk menampilkan profil, perjalanan akademik, dan proyek-proyek yang saya kerjakan sebagai seorang Full-Stack Web Developer.

## Tentang Saya
Saya adalah mahasiswa di Universitas Mataram yang memiliki minat mendalam di bidang pemrograman. Fokus utama saya adalah pengembangan perangkat lunak, mulai dari desain antarmuka web (HTML/CSS) hingga pemrograman backend menggunakan Java. 

Selain itu, saya memiliki ketertarikan khusus pada industri perbankan dan teknologi finansial (Fintech). Saya secara aktif mempelajari sistem Core Banking, khususnya Temenos T24 pada Modul AA (Arrangement Architecture), dengan tujuan untuk menggabungkan keahlian IT dan alur sistem perbankan.

## Keahlian & Teknologi
* **Frontend:** HTML, CSS (termasuk Flexbox dan implementasi desain responsif).
* **Backend:** Java (Object-Oriented Programming, pengkondisian logika, dan metode).
* **Cloud & Infrastruktur:** Saat ini sedang mendalami Amazon Web Services (AWS) meliputi layanan seperti EC2, S3, CloudFront, dan CloudWatch.
* **Lainnya:** Pembuatan sistem Point of Sale (POS) terintegrasi dan manajemen data operasional bisnis.

## Pengalaman & Proyek
* **Dicoding Scholarship Program:** Merancang dan membangun struktur website portofolio sebagai bagian dari kelulusan kelas Belajar Dasar Pemrograman Web.
* **AIQU Depot:** Mengembangkan dan mengelola sistem POS serta spreadsheet operasional untuk bisnis isi ulang air, dari tahap desain kupon hingga pencatatan transaksi.
* **Bale ATK:** Berpengalaman sebagai kasir, yang mengasah kemampuan manajerial, pelayanan pelanggan, dan ketelitian data.
* **Pengabdian Masyarakat:** Berpartisipasi aktif dalam program KKN di Sekotong Tengah, termasuk kegiatan pelestarian lingkungan seperti penanaman bibit mangrove.

## Cara Menjalankan Secara Lokal
1. Clone repositori ini ke dalam komputer lokal Anda.
2. Buka folder proyek menggunakan *code editor* (seperti VS Code).
3. Jalankan file `index.html` menggunakan ekstensi Live Server untuk melihat hasil akhirnya beserta seluruh interaktivitasnya.
